¿Qué es CSS?


Para poder entender de qué se trata el CCS Grid primero debemos entender y conocer qué es el CSS, esto es debido a que es gracias a esta tegnología/lenguaje que podemos ser capaces de darle estilo y color a nuestras páginas web, esto es básicamente porque como lo cita el siguiente video: El CSS es un lenguaje que facilita instrucciones para dar estilo a las etiquetas HTML como colores, tipos de letra, tamaño, etc.


Avila, M. L. (2020). OVI Aplicando Hojas de Estilo en Cascada CSS. [Objeto_virtual_de_Informacion_OVI]. Repositorio Institucional UNAD. https://repository.unad.edu.co/handle/10596/37679.

¿Qué es el CCS Grid?


Según Aguirre, H. J. O. Tesina:
El CSS grid se puede utilizar para lograr muchos diseños diferentes. Destaca por dividir una página en regiones principales, o definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.
Es decir, que el CCS Grid es un sistema que nos permite determinar una cuadrícula específica y modificar los nombres de los elementos html, para darle el tamaño, posición y colocación que querramos.


Aguirre, H. J. (15 de Diciembre de 2017). Repositorio de documentos digítales UPSIN. Obtenido de http://repositorio.upsin.edu.mx/formatos/Tesina2595.pdf
